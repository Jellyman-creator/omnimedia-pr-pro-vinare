# Omnimedia — PR pro vinaře (microsite)

Návrh microsite/landing page prezentující agenturu Omnimedia jako specialisty na PR pro vinaře.

- `index.html` — samostatná stránka ve vizuálu omnimedia.cz (font Montserrat, značkové barvy, hlavička i patička)

## Náhled
Otevři `index.html` v prohlížeči. Pokud je zapnuté GitHub Pages, web běží na adrese uvedené v nastavení repozitáře (Settings → Pages).
